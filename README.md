Đây là chương trình mô phỏng cấu trúc dữ liệu Cây Đỏ Đen qua giao diện dòng lệnh (Console).

## Cấu trúc dự án

- **`red_black_tree.py`**: Chứa toàn bộ thuật toán tạo cây, thêm nút, xoay cây và đổi màu.
- **`main.py`**: Chứa giao diện menu tương tác (nhập/xuất) dành cho người dùng.

## Cách chạy chương trình

1. Đảm bảo máy tính của bạn đã cài đặt **Python**.
2. Tải 2 file code về và để chung vào một thư mục.
3. Mở Terminal (trên Mac/Linux) hoặc Command Prompt / cmd (trên Windows) tại thư mục đó.
4. Gõ lệnh sau và nhấn Enter để chạy:
   ```bash
   python main.py
   ```
